Que es CSS y para que sirve

CSS es un lenguaje que se utiliza para definir el formato de los elementos del HTML

Las maneras de enlazar css a mi archivo html

Hay 3 formas de enlazar css al html:

Referenciando a otro archivo

Es decir crear otro archivo css y referenciarlo en head: "link rel="stylesheet" href="./styles.css"

Inline

Se agrega en la misma linea el atributo style a la etiqueta que puede ser h1, p

Con style dentro del archivo html

Agregando la etiqueta style en el head del documento, y dentro de este se agregaran las reglas de css
